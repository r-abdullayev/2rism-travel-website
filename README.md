# 2rism Travel Website

**2rism** is a responsive and modern travel website built using HTML, SCSS, and JavaScript. The project showcases a visually appealing front-end design, optimized for performance, accessibility, and user experience.

This website is intended for showcasing travel destinations, packages, and services. It’s structured to be easily extendable for integration with backend services or deployment as a static site.

## 🛠️ Tech Stack

- **HTML5** – Semantic and accessible structure  
- **SCSS** – Clean and maintainable styles with variables, nesting, and mixins  
- **JavaScript** – Adds interactivity and dynamic behaviors  
- **Vite** – Fast build tool and development server  
- **Git** – Version control
